Installation Wine

Vérifier si Wine est présent sur l'ordinateur et sa version 

wine --version

Wine est installé mais avec une version < 8.0

Désinstaller Wine 

sudo apt-get --purge remove wine

Si le paquet wine n'est pas trouvé : 

sudo apt-get --purge remove winehq

Ensuite : 

rm -rf $HOME/.wine

rm -f $HOME/.config/menus/applications-merged/*wine*

rm -rf $HOME/.local/share/applications/wine

rm -f $HOME/.local/share/desktop-directories/*wine*

rm -f $HOME/.local/share/icons/*wine* 

sudo apt-get update

Sur Ubuntu

Activer le mode 32 bit 

sudo dpkg --add-architecture i386

Créer le dépot 

sudo mkdir -pm755 /etc/apt/keyrings

sudo wget -O /etc/apt/keyrings/winehq-archive.key https://dl.winehq.org/wine-builds/winehq.key

Selon la version de Ubuntu :

  • Ubuntu 22.04 / Mint 21.x 
sudo wget -NP /etc/apt/sources.list.d/ https://dl.winehq.org/wine-builds/ubuntu/dists/jammy/winehq-jammy.sources
  • Ubuntu 20.04 / Mint 20.x 
sudo wget -NP /etc/apt/sources.list.d/ https://dl.winehq.org/wine-builds/ubuntu/dists/focal/winehq-focal.sources
  • Ubuntu 18.04 / Mint 19.x 
sudo wget -NP /etc/apt/sources.list.d/ https://dl.winehq.org/wine-builds/ubuntu/dists/bionic/winehq-bionic.sources

Sur Debian

Activer le mode 32 bit 

sudo dpkg --add-architecture i386

Créer le dépot 

sudo mkdir -pm755 /etc/apt/keyrings

sudo wget -O /etc/apt/keyrings/winehq-archive.key https://dl.winehq.org/wine-builds/winehq.key
  • Debian 11 
sudo wget -NP /etc/apt/sources.list.d/ https://dl.winehq.org/wine-builds/debian/dists/bullseye/winehq-bullseye.sources

Mettre à jour 

sudo apt update

Installer Wine 

sudo apt install --install-recommends winehq-devel

Installation Adobe DNG

Télécharger Adobe DNG 15.1 : https://www.adobe.com/go/dng_converter_win

Mettre Wine en Windows 10 et le configurer : 

WINEPREFIX="$HOME/wine-dng" winecfg

Dans Bibliothèques, copier dans Nouveau remplacement pour : 

api-ms-win-core-winrt-error-l1-1-0

Ajouter, puis Modifier et Désactiver

Installer Adobe DNG (modifier le chemin si la version a évolué) 

WINEPREFIX="$HOME/wine-dng" wine ~/Téléchargements/AdobeDNGConverter_x64_15_1_1.exe

À la fin de l'installation sur Lancer Adobe DNG Converter pour vérifier que tout s'est bien passé. Si le logiciel s'ouvre correctement 

echo "alias dng='WINEPREFIX=\"\$HOME/wine-dng\" wine \"\$HOME/wine-dng/drive_c/Program Files/Adobe/Adobe DNG Converter/Adobe DNG Converter.exe\"'" >> ~/.bashrc && exec bash

Pour ouvrir Adobe DNG Converter, taper dans la console dng
